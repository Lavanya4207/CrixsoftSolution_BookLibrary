# 📚 Personal Book Library

## Description
Personal Book Library is a web application developed using HTML5, CSS3, and JavaScript. It helps users manage their personal book collection.

## Features
- Add New Books
- Search Books
- Book Categories
- Borrow & Return Books
- Edit Book Details
- Delete Books
- Borrowing History
- Local Storage Support

## Technologies Used
- HTML5
- CSS3
- JavaScript
